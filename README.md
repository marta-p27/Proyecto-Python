# Proyecto-Python
Primer proyecto en Python cuyo fin es crear un organizador de estudio básico.

Según el plan inicial:
En este proyecto de Pyhton el objetivo es crear un asistente que, después de serle introducido tu horario, horas en las que te es posible estudiar y calendario de exámenes y trabajos, te recomienda cómo deberías organizarte para estudiar y qué horas dedicarle a la preparación de cada examen. 
El asistente pedirá la elección del horario, de forma que este se cargue y los exámenes pendientes. Así, podrás consultar las asignaturas del día/día siguiente (según la hora que sea), los exámenes pendientes, cuántos dían quedan para ellos... Te dará el tiempo aproximado que deberás dedicarle y te ordenará en un calendario su recomendación de cuándo deberías estudiar.
